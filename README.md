# Android-Led-Controller-with-Arduino
Simple led controller using Arduino on Android

## Components
* Arduino UNO/NANO
* LED
* Resistor 221 ohms
* Cellphone
* HC-05/06

## Connections
Arduino Pins | Bluetooth Pins
------------ | -------------
RX (Pin 0) | TX
TX (Pin 1) | RX
5V | VCC
GND | GND
